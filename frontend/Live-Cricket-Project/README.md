## Features

- User authentication for personalized experience
- Real-time cricket match updates
- Comprehensive features list showcasing all functionalities
- Responsive design for mobile and desktop users

## Installation

To get started with the Live Cricket Project, follow these steps:

1. Clone the repository:
   ```
   git clone https://github.com/Shohag2100/Live-Cricket-Project.git
   ```
2. Navigate to the project directory:
   ```
   cd Live-Cricket-Project
   ```
3. Install the dependencies:
   ```
   npm install
   ```

## Usage

To run the application locally, use the following command:
```
npm start
```
The application will be available at `http://localhost:5173`. After signing in, users will be redirected to the home page displaying all features.
